# FastGit 繁體中文指南

感謝您閱讀 FastGit 中文指南，請在側欄選擇需要的文檔。

## 關於 FastGit

FastGit 是一個對於 GitHub.com 的鏡像加速器。我們使用開放資源為 GitHub 加速。

## 提醒

FastGit 目前仍處於測試階段。我們會根據實際情況調整服務。

檔案站未即時更新。有關最新檔案，請參閱此[連結](https://github.com/fastgitorg/document)。
