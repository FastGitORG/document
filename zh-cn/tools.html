# 辅助工具

为方便 FastGit 的使用，我们构建了如下官方工具。具体信息请参阅对应项目文档。

# 油猴脚本

油猴脚本用于辅助用户使用 FastGit 的服务。用户可以通过 GreaseMonkey、TamperMonkey 等插件使用下列脚本。

项目 地址 主要维护人员
FastGit.user.js https://gist.github.com/vhxubo/d67fbd5bb3b7308b2e3690ca58e12c12 (opens new window), https://greasyfork.org/zh-CN/scripts/402301-fastgit (opens new window) @vhxubo (opens new window)
GitHub_with_FastGit.user.js https://github.com/FirefoxBar/userscript/tree/main/GitHub_with_FastGit (opens new window) @ShuangYa (opens new window)

# PSFastGet

一个 Powershell 脚本用于下载 GitHub 文件。目前已被 fgit 所取代。

项目 地址 主要维护人员
